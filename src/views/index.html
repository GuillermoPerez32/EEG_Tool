EEG2BIDS Wizard is a tool for de-identification of EDF data and conversion to BIDS format for data sharing. This software is designed to run on EDF files (EEG or stereo iEEG) for one recording from one subject at a time. Events and metadata such as a LORIS ProjectID and Visit Label can be included. For more information about BIDS, bids.neuroimaging.io
